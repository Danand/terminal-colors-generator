# Terminal Colors Generator

```bash
$ python -m terminal_colors_generator --help

usage: Generates new color scheme from given color scheme.

stdin: Colors JSON object.

options:
  -h, --help            show this help message and exit
  --color-from COLOR_FROM
                        Color which considered to be initial value for color wheel rotation
  --color-to COLOR_TO   Color which considered to be destination value for color wheel rotation
  --export              Outputs Bash variable exports.
```
